# T-Rex Selector Methods — Python Demos

Python demos for the T-Rex selector family, using the `trex_selector_neo`
package. Currently one variant is available in Python:

| Variant | Folder | Status |
|---|---|---|
| Classical T-Rex | [trex/](trex/) | Available — Demos 01–06 (single runs, Monte Carlo simulations, L-loop strategies, memory-mapped pipelines) |

---

## Not Yet Ported

The following T-Rex variants exist in the
[cpp/trex_selector_methods/](../../cpp/trex_selector_methods/) tree but are
**not yet available in Python** — see cpp/ or R/ for those:

- `trex_da/` — Dependency-Aware T-Rex
  ([cpp](../../cpp/trex_selector_methods/trex_da/), `R/trex_selector_methods/trex_da/`)
- `trex_gvs/` — Grouped Variable Selection T-Rex
  ([cpp](../../cpp/trex_selector_methods/trex_gvs/), `R/trex_selector_methods/trex_gvs/`)
- `trex_spca/` — Sparse PCA with T-Rex
  ([cpp](../../cpp/trex_selector_methods/trex_spca/), `R/trex_selector_methods/trex_spca/`)
- `trex_screening/` — Ultra-high-dimensional screening
  ([cpp](../../cpp/trex_selector_methods/trex_screening/), `R/trex_selector_methods/trex_screening/`)
