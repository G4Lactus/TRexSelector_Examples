# T-Rex DA

## The Conditional Penalty Factor — Open Theoretical Problem for Future Work

### Motivation

Under AR(1) with well-separated true actives, the original group
$\text{Gr}(j, \rho_{\text{thr}})$ contains only correlated null neighbors of $j$.
These nulls accumulate moderate relative occurrences $\Phi(j') > 0$ through
their correlation with the true active, driving $\delta \approx 0$ and
$\Psi \approx 1/2$ — the maximum penalty fires on the true active itself.
The conditional group addresses this by restricting penalization to
**competitive candidates** only.

### Definition

For a fixed baseline threshold $v_0 \in [0.5, 1)$, the conditional group is:

$$
\text{Gr}^*(j, \rho_{\text{thr}}, v_0)
= \bigl\{j' \in \text{Gr}(j, \rho_{\text{thr}}) : \Phi_{T,L}(j') > v_0\bigr\},
$$

and the conditional penalty factor is:

$$
\Psi^*_{T,L}(j, \rho_{\text{thr}}, v_0) \;=\;
\begin{cases}
\dfrac{1}{2 - \displaystyle\min_{j' \in \text{Gr}^*(j,\rho_{\text{thr}},v_0)}
|\Phi_{T,L}(j) - \Phi_{T,L}(j')|},
& \text{Gr}^*(j, \rho_{\text{thr}}, v_0) \neq \emptyset \\[14pt]
1,
& \text{Gr}^*(j, \rho_{\text{thr}}, v_0) = \emptyset
\end{cases}
$$

The DA relative occurrence is then $\Phi^{DA,*}_{T,L}(j, \rho_{\text{thr}})
:= \Psi^*_{T,L}(j, \rho_{\text{thr}}, v_0) \cdot \Phi_{T,L}(j)$,
and the FDP estimator in Definition 5 is re-evaluated with $\Phi^{DA,*}$
replacing $\Phi^{DA}$ everywhere — no structural change to Algorithm 1,
only the group lookup changes.

The two structural changes from Definition 1 are:

1. The group is intersected with $\{j' : \Phi_{T,L}(j') > v_0\}$,
   restricting penalization to variables that are themselves competitive candidates.
2. The empty-group case returns $\Psi^* = 1$ instead of $1/2$: if no competitive
   neighbor exists, variable $j$ is not penalized at all.

### This Construction is Group-Agnostic

The conditional filter $\Phi_{T,L}(j') > v_0$ operates as a post-processing
step on whatever base group $\text{Gr}(j, \rho_{\text{thr}})$ is supplied.
The subset monotonicity of Theorem 3 is inherited unchanged, since
$\mathcal{C}_{v_0} = \{j' : \Phi_{T,L}(j') > v_0\}$ does not depend on
$\rho_{\text{thr}}$.

The base group itself is model-specific. The general form is direct
correlation thresholding:

$$
\text{Gr}(j, \rho_{\text{thr}}) =
\{j' \in \{1,\ldots,p\}\setminus\{j\} : c(j, j') \geq \rho_{\text{thr}}\},
$$

where $c(j,j')$ is a model-determined similarity measure:

| Setting | $c(j, j')$ | Requires estimation? |
| :-- | :-- | :-- |
| AR(1): known $\rho$ | $\rho^{\|j - j'\|}$ | No — closed form |
| AR(1): unknown $\rho$ | $\hat{\rho}^{\|j - j'\|}$ | Yes — $\hat{\rho}$ from WLS |
| AR($q$) | $\hat{\rho}(\|j-j\|)$ from Yule-Walker | Yes — ACF estimation |
| General | $\|\hat{\rho}_{j,j'}\|$ from sample $\hat{\mathbf{R}}$ | Yes — full correlation matrix |

The paper's dendrogram (Definition 2) is **not** a direct correlation threshold.
It defines groups recursively through cluster merge distances, which generates
the required subset monotonicity by construction of the binary tree.
The conditional filter applies to it unchanged — only the group membership
oracle differs.

## Unified Formulation

The completely general conditional DA group that covers AR(1), AR($q$), the paper's block model,
and arbitrary covariance structures is:

$$
\boxed{
\text{Gr}^*(j, \rho_{\text{thr}}, v_0) = \bigl\{j' \in \{1,\ldots,p\}\setminus\{j\} \;:\; |\hat{\rho}_{j,j'}| \geq \rho_{\text{thr}} \;\text{ and }\; \Phi_{T,L}(j') > v_0\bigr\}
}
$$


### The Open Problem

The construction introduces a fundamental difficulty:
$\Phi_{T,L}(j')$ is computed from the same $K$ random experiments that
produce $\Phi_{T,L}(j)$. The group $\text{Gr}^*$ is therefore a
**random, data-dependent set**, measurable with respect to the
$\sigma$-algebra generated by all $K$ experiments.

The FDR control proof in Theorem 2 relies on the super-martingale property of
$\{H_{T,L}(v, \rho_{\text{thr}})\}_v$ stated in Lemma 1, which was established
under the assumption that the groups $\text{Gr}(j, \rho_{\text{thr}})$ are
**deterministic** — fixed before any random experiment is run.
With a data-dependent $\text{Gr}^*$, the super-martingale property is no
longer guaranteed and the FDR proof does not carry through.

Fixing $v_0 = 0.5$ does not resolve this: it makes $\text{Gr}^*$ adapted to
the filtration, but adaptedness is necessary — not sufficient — for Doob's
optional stopping theorem. The super-martingale property of $H_{T,L}$ must
be re-established from scratch under a data-dependent group.

### Two Principled Fixes

**Fix 1 — Sample splitting.**
Partition the $K$ experiments into two independent halves $K_1$ and $K_2$.
Use $K_1$ to compute pilot relative occurrences $\Phi^{(1)}$ and form:

$$
\text{Gr}^*(j) = \bigl\{j' \in \text{Gr}(j, \rho_{\text{thr}}) :
\Phi^{(1)}_{T,L}(j') > v_0\bigr\}.
$$

Then use $K_2$ exclusively for $\Phi^{(2)}_{T,L}(j)$ in the FDP estimator.
Since $\text{Gr}^*$ is independent of the $K_2$ experiments, it is
effectively deterministic with respect to the martingale filtration,
and Lemma 1 carries through.
Cost: power reduction from $K_2 < K$.

**Fix 2 — Two-stage procedure.**
Run a preliminary T-Rex pass with $K_{\text{pre}} \ll K$ experiments.
Form $\text{Gr}^*$ from the resulting pilot relative occurrences as a
**fixed, pre-computed group**, independent of the main $K$-experiment run.
The main run then uses the fixed $\text{Gr}^*$ exactly as an ordinary
deterministic group — the FDR proof is intact.
Cost: one additional forward-selection pass of complexity
$\mathcal{O}(np \cdot K_{\text{pre}})$.
